# Home Advisor Vision Review — Strategic Recommendations

## Executive Summary

After reviewing:
- the Vision UI
- orchestration architecture
- uploaded screenshots
- enhancement workflows
- simplification reset
- current implementation files
- Codex project review

my conclusion is:

# Home Advisor should pivot away from being an AI image transformation platform
and reposition Vision as a:
# Seller Photo Readiness Assistant

This is the correct strategic direction.

---

# Main Problems

## 1. The UI Is Overwhelming

The current Vision experience exposes:
- stages
- orchestration
- workflow states
- enhancement ladders
- fallback terminology
- marketplace readiness
- confidence systems
- pipeline summaries

This feels like:
- an internal AI tool
NOT:
- a seller product

---

## 2. AI Results Are Not Strong Enough

The current Medium Declutter outputs are:
- technically safer
- structurally cleaner
- less broken

BUT:
- visually weak
- not persuasive
- not useful enough

The system is spending enormous effort on orchestration and scoring while producing barely noticeable improvements.

---

## 3. Weak AI Results Are Being Presented As Success

Examples:
- Safe Enhancement Fallback
- subtle preview generated
- deterministic fallback returned

These are engineering outcomes, not customer value.

If enhancement quality is weak:
- do not pretend it succeeded
- do not expose fallback terminology
- provide seller guidance instead

---

## 4. The Product Is Trying To Solve Too Much

Vision currently attempts:
- declutter
- staging
- repainting
- flooring
- room clearing
- remodeling
- listing-safe rendering
- orchestration
- QA ranking

This is too broad.

---

# What Is Actually Strong

The project itself is NOT failing.

Several parts are genuinely strong:

## Seller Guidance
The system already understands:
- clutter
- lighting
- buyer appeal
- room usefulness
- furniture density
- retake recommendations

This is valuable.

## Marketing Outputs
Your:
- flyers
- reports
- comps
- seller summaries
- marketing package generation

are more valuable than the current AI editing.

## Deterministic Enhancement
The local enhancement pipeline is:
- fast
- believable
- trustworthy
- stable

This should become the core Vision feature.

## Window Protection
The newer architecture preserves:
- windows
- geometry
- perspective

This is real progress.

---

# Strategic Direction Reset

Vision should become:

# Seller Photo Coach

NOT:
# AI Remodeling Engine

---

# Recommended Product Experience

Each photo gets ONLY:

## Improve Photo
Fast deterministic enhancement.

## Clean Up Photo
Moderate clutter cleanup only.

## Explore Ideas
Optional concept previews.

No workflow ladders.
No stages.
No orchestration language.

---

# Seller Guidance Should Become Central

Examples:

- Remove throw blanket
- Simplify shelf decor
- Open blinds evenly
- Retake from wider angle

This is:
- useful
- trustworthy
- actionable
- realistic

Often more valuable than weak AI edits.

---

# Important Product Truth

# Bad AI is worse than no AI.

If the enhancement barely changes the room:
DO NOT:
- pretend it succeeded
- show fallback terminology
- expose orchestration logic

Instead:
- keep original photo
- give guidance
- recommend retake improvements

---

# Recommended Product Strategy

## Phase 1 — Stabilize

Focus ONLY on:
- Improve Photo
- Seller Guidance
- Best Photo Selection
- Flyer / Report Generation

This is already valuable.

## Phase 2 — Cleanup

Add:
- lightweight declutter
- cleanup-only workflows

Avoid:
- reconstruction
- aggressive furniture removal
- remodeling

## Phase 3 — Concepts

Keep:
- floors
- walls
- kitchen previews
- open room concepts

BUT:
- hide behind “Explore Ideas”
- clearly label as concepts
- never market as listing-ready

---

# Build vs Buy Recommendation

My recommendation:

# Stop building advanced generative editing internally for now.

Furniture removal, flooring replacement, and remodeling are:
- extremely difficult
- expensive
- inconsistent

This is a specialized AI company problem.

---

# Strong Recommendation

Run provider bakeoffs using:
- Virtual Staging AI
- Decor8 AI
- HomeDesigns AI
- Adobe Firefly
- Cloudinary AI
- Photoroom
- Stability
- OpenAI image editing

Only integrate if results are dramatically better.

---

# Product North Star

The seller should feel:

“I uploaded photos and immediately got:
- better-looking images
- useful preparation advice
- stronger marketing materials
- a clearer plan.”

NOT:

“I entered a complicated AI workflow.”

The project is NOT failing.

But the Vision module became:
- too ambitious
- too workflow-heavy
- too AI-centric
- too internally focused

The correct move now is:
- simplification
- trust
- speed
- usefulness
- seller-first value

This gives Home Advisor:
- a cleaner identity
- a more reliable workflow
- stronger UX
- clearer market positioning
