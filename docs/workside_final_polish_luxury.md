# 🎯 1. TYPOGRAPHY SYSTEM (CRITICAL)

## PROMPT 1 — STANDARDIZE TYPE SCALE

```text
Task:
Implement strict typography hierarchy.

Rules:

H1 (Page Titles):
- Large, bold
- Tight letter spacing
- Example: "Peralta Residence"

H2 (Section Headers):
- Medium weight
- Clear spacing above

H3 (Subsections):
- Slightly smaller
- Consistent weight

Body:
- Comfortable line height (1.5–1.7)
- No dense paragraphs

Numbers (scores, ROI):
- Bold but NOT oversized
- Must not dominate layout

Goal:
Clear hierarchy without visual noise
```
