# Workside Home Advisor — UX Copy Spec (Codex-Ready)

Version: 1.0

Purpose:
Provide exact UX copy for Seller and Realtor guided workflows, sidebar wizard, checklist, mobile capture, provider prompts, and system states.

Tone:
- Clear
- Short
- Action-oriented
- Confidence-building
- No jargon

---

# GLOBAL TONE RULES

- Use simple language
- One action per step
- Avoid long explanations
- Always guide next action
- Reinforce progress

Examples:
✔ “Add your property”
✔ “Upload photos”
✖ “Please proceed to initiate the property creation workflow”
