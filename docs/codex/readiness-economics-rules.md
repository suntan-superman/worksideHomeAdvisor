# Readiness Economics Rules

## Structure
The readiness economics section now uses:
1. one headline metric
2. one plain-language explanation
3. one interpretation line
4. one risk reminder

## Anti-repetition
- Remove repeated ROI phrasing across callouts and bullets.
- Keep economics concise and decision-oriented.

## Behavior by class
- Prep/balanced: include economics by default
- Launch: include when unresolved risk, blockers, or price uncertainty still need context
