# Flyer Copy Guidelines

Copy updates now prioritize:

- readiness-aware tone
- concise, property-grounded language
- conversion-oriented but credible statements

Rules:

- Do not overstate readiness in preview mode.
- Keep wording specific to available property data and selected photos.
- Avoid unsupported superlatives and certainty claims.
- Keep subheadline and summary concise and buyer-facing.

Implementation behavior:

- `preview` mode forces preview-safe fallback copy.
- `launch_ready` and `premium` modes can use marketing copy when present.
- Fallback copy remains available if AI marketing output is missing.
