# Flyer Class Application

## Peralta-like state
- Low readiness
- Price pending
- Zero marketplace-ready photos
- Result: `preview_flyer` with gallery/pacing gates

## Mainsail-like state
- Mid/high readiness
- Chosen price confirmed
- Multiple marketplace-ready photos
- Result: `prelaunch_flyer` or `marketing_flyer` depending on readiness + gallery strength

## Objective
Avoid rendering both properties as the same flyer shell with swapped text.
