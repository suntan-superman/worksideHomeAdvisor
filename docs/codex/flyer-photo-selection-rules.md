# Flyer Photo Selection Rules

Flyer photo selection now uses enhanced ranking and balance logic:

1. Score photos by:
  - quality score
  - listing-candidate status
  - preferred variant availability
  - room-priority boost
  - retake penalty
2. Honor manual selections first when provided.
3. Enforce balanced room coverage where possible:
  - exterior
  - kitchen
  - living/main area
  - bedroom
  - bathroom
4. Avoid over-repeating the same room bucket when alternatives exist.
5. Limit to top 4 selected photos.

This selection is used before flyer mode scoring so weak sets naturally downgrade to preview framing when appropriate.
