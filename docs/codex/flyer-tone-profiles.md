# Flyer Tone Profiles

## `preview_flyer`
- Honest preview language
- Minimal listing-grade claims
- CTA supports early inquiry while prep completes

## `prelaunch_flyer`
- Persuasive but measured
- Uses stronger feature and gallery sequencing where available

## `marketing_flyer`
- Sales-forward, listing-grade voice
- Visual-first narrative and showing CTA

## Source
Flyer tone profiles are defined in `output-class-engine.js` and used by flyer rendering in `html-pdf.service.js`.
