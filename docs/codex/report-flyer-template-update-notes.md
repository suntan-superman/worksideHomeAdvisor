# Report/Flyer Template Update Notes

Updated template behavior:

- Report provider cards now show source labels and confidence notes.
- Report recommendation block prefers structured recommendation-action lines when available.
- Report seller-guidance section now includes consequence framing summary.
- Report ROI block now surfaces decision-driven message and net-upside badge.
- Provider empty state copy now indicates both marketplace and nearby discovery outcomes.

- Flyer template now reads mode (`preview`, `launch_ready`, `premium`) and renders mode-aligned hero signal text.
- Flyer CTA card and CTA buttons now pull from structured CTA metadata labels.
- Flyer cover badges can include mode label.
- Flyer fallback PDF includes mode support text and CTA metadata fallback text.

Goal preserved:

- Keep existing visual direction while improving clarity, actionability, and readiness alignment.
